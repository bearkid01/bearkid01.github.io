Data Visualization Project

Planned Tools

Besides D3.js, I might use PostgreSQL's "group by" function to generate some aggregated numbers.

Planned Techniques

  • Dashboard: I plan to use a dashboard to display the following three techniques.
  • Symbol Map: The first step is to generate/obtain a NYC TopoJSON file (with neighborhood boundaries preferred). The locations of incidents will be shown in bubbles/circles filled with color. The columns that will be used to encode the locations are the two columns in the dataset containing latitudes and longitude of the incidients.
  • Line Plot: It can be used to display the number of incidents for a specific neighborhood overtime. When a user click on a neighborhood in the Symbol Map, the line plot will display the time series of the number of incidents for that neighborhood. If none of the neighborhoods in NYC is clicked, the line plot will display the time series of the number of incidents aggregated from all the neighborhoods.
  • Bar(Stacked) Plot: It can be used to display the number of incidents by location type for a specific region. When users click on a neighborhood in the Symbol Map, the bar plot will display the count of incidents by location type, such as Restaurant, Cafeteria or Food Cart. If none of the neighborhoods in NYC is clicked, the bar plot will display the counts of incidents by location type aggregated from all the neighborhoods.

Planned Interaction

  • Zooming: Users can scroll mouse to zoom in to a neighborhood.
  • Filtering: Users can filter the map based on Status and Complaint Type.
