Motivation

For C-level functionality, I focused on applying the basic functionality that I have learned so far, including creating circles, positioning them, filling them with colors, creating x and y axes, setting labels for the axes, adding tooltips and a legend. For B-level and A-level functionality, I wanted to apply more advanced visualization techniques. For example, creating a scatterplot matrix can help me visualize the pairwise relationship between a set of variables. I found that the illteracy rate and life expectancy reversely correlated. Besides, parallel coordinates plot is a great way to visualize high dimension data in 2D space. Since this data set has more than two dimensions, I used parallel coordinates to discover the underlying patterns among regions.
