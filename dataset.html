Data Visualization Project

Source

The dataset I chose to visualize is the food poisoning dataset from NYC OpenData. Click here to be directed to the source page. I also include the CSV-formatted data in the repository.

Purpose

The main reason to choose this dataset to visualize is because it has both timestamps and geographical locations (longitude and latitude), which gives me the options of visualizing it in time series or on a map or both. Besides, there are some missing values in the dataset. I want to take this opportunity to practice how to handle missing values for data visualization.

Types of Variables

  • Timestamp: Created Date, Closed Date
  • Categorical: Complaint Type, Location Type, Status
  • Numerical: X Coordinate (State Plane), Y Coordinate (State Plane)
  • Text: Incident Address
